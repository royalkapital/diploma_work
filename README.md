# diploma_work
## Dataset

This project uses the DARPA Operationally Transparent Cyber (OPTC) dataset,specifically the ECAR (Event Collection and Analysis Record) logs.
The dataset is large (approximately 6–7 GB) and cannot be stored directly  in the GitHub repository due to file size limitations.

You can download the dataset from the following link:

https://drive.google.com/file/d/1rngYdnYkN4zba-Bd4zIKEZsK879JIQTr/view?usp=sharing
